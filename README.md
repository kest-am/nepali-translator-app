# Nepali Translator

A web app that transcribes and translates Nepali audio using Google's Gemini AI.

## Requirements

- Docker Desktop
- Git

## Installation

Clone the repository:

```bash
git clone https://github.com/kest-am/nepali-translator-app.git
cd nepali-translator-app
```

Build and start the app:

```bash
docker compose up --build
```

When the server starts, open:

http://localhost:3000

Stop the app with:

```bash
Ctrl + C
```

To start it again later:

```bash
docker compose up
```
